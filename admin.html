Admin local

Important : cette page fonctionne sans Cloudflare et sans base de données. Elle sert à préparer le contenu facilement. Pour publier sur Vercel, clique Exporter site-content.json, puis upload ce fichier dans GitHub à côté de index.html. Vercel va redéployer automatiquement.

Ce n’est pas encore une admin sécurisée qui écrit directement dans le site. C’est la meilleure version simple tant qu’on reste sur Vercel sans backend.

Ajouter une photo de projet

Tu peux choisir plusieurs photos pour le même projet. Le site ouvrira un album swipe.

Ajouter un avis affiché

Pour l’instant, ce sont des avis que tu contrôles toi-même. Plus tard, on pourra faire des avis clients réels avec base de données.

Exporter pour publier

Quand tu as fini d’ajouter tes photos, avis ou articles, exporte le fichier. Ensuite, upload site-content.json dans ton repo GitHub, au même endroit que index.html. Vercel va redéployer le site.

Les données restent sauvegardées dans ton navigateur pour continuer plus tard sur le même appareil.
